# Windows Server & Active Directory Lab

Hands-on lab work covering Windows Server administration, Active Directory configuration, Linux access control, and security hardening. Labs include real configuration tasks performed in a virtualized Windows Server environment with an Active Directory domain, a Linux server, and endpoint hardening scenarios.

---

## Environment

- **Windows Server:** Active Directory Domain Services, Group Policy, Task Scheduler, Event Viewer
- **Linux Server:** Ubuntu (user/group management, chmod, sticky bit)
- **Tools:** Active Directory Users and Computers, Group Policy Management Console, Nmap, IIS Log Viewer, Windows Event Viewer

---

## Lab Modules

| File | Description |
|------|-------------|
| `active-directory-configuration.md` | Task Scheduler backup automation, Event ID 4624 logon trigger, OU creation, user provisioning |
| `access-control-rbac.md` | Role-change account modification, GPO password policy, Linux account creation, connection to M365 lifecycle automation |
| `linux-permissions.md` | chmod with octal notation, sticky bit, least privilege, user/group management, log analysis, port closure |

---

## Skills Demonstrated

- Active Directory OU design and user provisioning
- Group Policy configuration and enforcement
- Task Scheduler automation (backup and event-triggered tasks)
- Windows event log analysis (Event ID 4624)
- Linux file permission management (chmod, octal notation, sticky bit)
- Linux user and group administration
- Log analysis across platforms (Event Viewer, IIS, Linux)
- Port discovery and closure with Nmap
- RBAC principles applied across Windows and Linux environments
